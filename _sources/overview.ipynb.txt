    "This shows the names, shapes, and dtypes of all the tensors starting from the root tensor (in this case, `loss`). It also shows the operations which took place to create the root. In theory, we can make computation graphs as complex as we want to as long as it results in a 1 element Tensor as the root.\n",
    "\n",
    "Note that the `2` in the *2 operation gets turned into a scalar Tensor (with a random name).\n",
    "\n",
    "Now for the important part: the gradient. Using SimpleTensor, we can automatically calculate the gradient $\\frac{\\partial \\text{loss}}{\\partial a_{i,j}}$. Since loss is a scalar (it should be), we can calculate the matrix derivative, or Jacobian, of loss with respect to the elements of a.\n",
    "\n",
    "In the `sum` operation, we have $\\text{loss}=2\\sum a_{i,j}$. If we take any derivative $\\frac{\\partial \\text{loss}}{\\partial a_{i,j}}$, we can see that it's a 2 for every element.\n",
    "\n",
    "Let's see this automatically calculated using the `backward` function."
